## 🌱 快速開始

這門課有 11 個課程，涵蓋打造 AI Agents 的基礎知識。每個課程都有自己的主題，想從哪裡開始都可以！

這門課有多語言支援。請到我們的[支援語言列表](../..)查看。

如果你是第一次使用 Generative AI 模型來開發，可以參考我們的[Generative AI For Beginners](https://aka.ms/genai-beginners)課程，裡面有 21 個課程教你如何用 GenAI 開發。

別忘了幫這個 repo [點星 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)並且[分支這個 repo](https://github.com/microsoft/ai-agents-for-beginners/fork)來執行程式碼。

### 你需要準備什麼

這門課的每個課程都包含程式碼範例，可以在 code_samples 資料夾找到。你可以[分支這個 repo](https://github.com/microsoft/ai-agents-for-beginners/fork)來建立自己的副本。

這些練習中的程式碼範例，會使用 Azure AI Foundry 以及 GitHub Model Catalogs 來與語言模型互動：

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - 免費 / 有限制
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 帳號

這門課也會使用 Microsoft 提供的以下 AI Agent 框架與服務：
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)

欲了解更多關於本課程程式碼執行的資訊，請參考 [Course Setup](./00-course-setup/README.md)。
